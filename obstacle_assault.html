Obstacle Assault Game

Action-packed obstacle dodging game built with Unreal Engine.

Gameplay Demo

Overview

"Obstacle Assault" is a thrilling game where players navigate through challenging levels filled with unique obstacles, testing their timing and precision in an engaging, fast-paced environment.

Key Features

  • Dozens of handcrafted levels with increasing difficulty.
  • Realistic physics-based mechanics.
  • Engaging soundtrack to enhance the experience.

Development Tools

  • Unreal Engine for level design and gameplay logic.
  • Blueprints and C++ for programming.
  • Photoshop for UI/UX design.
