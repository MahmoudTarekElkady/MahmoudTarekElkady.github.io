Color Swap Mobile Puzzle Game

A portrait-oriented mobile puzzle game built in Unity involving color swaps and graph theory constraints.

Gameplay Demo

Details

Color Swap is a clean, minimal puzzle game where players swap node colors on a graph to solve connection conflicts. The rule is simple: no two connected nodes may share the same color.

Key Features

  • Tactile Mechanics: Responsive drag-and-drop controller allowing users to grab and swap node positions.
  • Graph Coloring Validation: Lightweight state machine validation algorithm tracking the states of connected edges. Conflict lines dynamically highlight in red.
  • Dynamic Board Generation: Levels generate dynamically at runtime using modular ScriptableObject data files.
  • DOTween Animations: Smooth animations built for node grabs, snaps, target moves, and victory screens.

Tools Used

  • Unity Engine
  • C# Scripting (Systems & Graph Logic)
  • DOTween (UI Animations & Motion Layouts)
