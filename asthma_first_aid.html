Asthma Attack First Aid
VR Experience

Interactive safety and medical training for handling severe asthma attacks.

Watch Walkthrough

Details

This project focuses on teaching critical life-saving skills in a controlled virtual environment by:

  • Recreating high-stress scenarios to build muscle memory for first aid.
  • Guiding the user interactively through the steps to stabilize a patient with an asthma attack.
  • Utilizing a highly interactive XR environment for engaged learning.

Tools Used

  • Unity Engine
  • XR Interaction Toolkit
  • C#
